# PLAN

Create a react app which sends requests to the Open Move Database to retrieve movie information based on a user search query, and display all the relevant information for that film on screen.

## Components Needed

- Input

  - Text Field
  - Submit Button

- Movie Display Card
  - Title
  - Year
  - Director
  - Poster
  - Actors
  - Short Plot

### App

- State

  - Movie

- DisplayMovie()
  - GetDataFromApi()

### Input

- State

  - Text

- Props
  - DisplayMovie()

### Movie Display

- Props
  - Title
  - Year
  - Director
  - Poster
  - Actors
  - Short Plot
